🤝 거래 방식 비교

방식 A. 상차가 제시 — 판매자가 상차까지
구매자 지불 = 나무값 + 굴취비 + 상차비
구매자는 운송비만 별도 부담
장점: 단가를 높게 받을 수 있음
단점: 장비/인부 섭외 + 작업 관리 필요
방식 B. 목대가 제시 추천
구매자 지불 = 순수 나무값만
구매자가 자기 장비로 직접 굴취+상차+운송
장점: 판매자 신경 쓸 일 없음, 상차비 부담 없음
일괄 매도(밭떼기)에서 일반적인 방식
매물 등록 시 가격 표기 예시
목대가(나무값만): 그루당 ○○만 원~
  → 굴취/상차는 구매자 부담
상차가(굴취+상차 포함): 그루당 ○○만 원~
  → 운송비만 별도
일괄 매도(밭떼기) 협의 가능
추천 전략
  • ● 목대가(방식B) 기준 판매가 현실적
  • ● 조경업체/유통상은 자체 장비 보유 → 직접 작업 선호
  • ● 매물 등록 시 목대가+상차가 두 가지 모두 표기 권장
