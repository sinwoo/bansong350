## 판매 채널 구조

### 1차 채널 (협회/공공)
```
한국조경수협회 본회 ─┬─ 경북(대구) 지회 → 지역 업체 연결
                    └─ 매입의뢰 게시판 → 전국 회원 노출
산림조합중앙회 ────── 대구경북본부 → 시세 감정 + 매수자 연결
```

### 2차 채널 (온라인 플랫폼)
```
트리디비 (2만 회원, 30% 점유) ─┐
나무시장 (kotr.co.kr)         ├─ 전국 구매자 노출
나무몰 (namoomall.com)        │
라펜트 (전문가 포털)           │
마켓트리                       │
산림조합 직거래장터             ┘
```

### 3차 채널 (직접 연락)
```
유통상/중개업체 (8곳) → 대량 매입 가능
경북/대구 시공업체 (5곳) → 현장 접근성 우수
전국 대형 시공업체 (8곳) → 대규모 수요
```
