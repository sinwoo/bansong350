## 판매 시나리오

### SC-001: 협회 경유 매수자 연결

**시나리오**:
매도인이 한국조경수협회에 연락하여 매수자를 소개받는다.

#### 정상 흐름
- Given: 조경수협회 연락처 확보
- When: 본회/경북지회에 전화하여 매물 안내
- Then: 회원 업체 중 매수 의향자 소개받음

#### 엣지 케이스
- [ ] 협회에서 비회원 매물 취급 거부 시 → 회원 가입 또는 지회장 직접 접촉
- [ ] 소개받은 업체가 현장 방문 후 가격 크게 깎을 때 → 복수 견적 비교 후 대응
- [ ] 시즌 후반이라 수요 감소 시 → 가격 유연하게 조정

### SC-002: 온라인 플랫폼 매물 등록

**시나리오**:
트리디비 등 플랫폼에 매물을 등록하여 전국 구매자에게 노출한다.

#### 정상 흐름
- Given: 매물 사진 및 정보 준비 완료
- When: 각 플랫폼에 회원가입 후 매물 등록
- Then: 구매자로부터 문의 연락 수신

#### 엣지 케이스
- [ ] 등록 후 문의 없을 때 → 사진 교체, 가격 조정, 프리미엄 등록
- [ ] 사기 의심 연락 → 업체 실체 확인 (사업자등록, 시공 실적)
- [ ] 여러 구매자 동시 관심 → 최고가 또는 조건 비교 후 선택

#### 정상 흐름
- Given: 유통상이 현장 방문하여 수목 확인
- When: 일괄 매입 가격 제시
- Then: 가격 협상 후 계약

#### 엣지 케이스
- [ ] 제시가가 원가 이하 → 거절 후 다른 업체 대기

- [ ] 굴취비 분담 이슈 → "상차가" 기준 명확히 합의
